# NIC VALIDATOR

Simple api to validate and get details of sri lankan nic,
So I will not have to code this same thing again

## response

```
{
    "birthday": {
        "year": 1997,
        "month": 8,
        "day": 29
    },
    "gender": "Male"
}
```

You can access api here : https://nic-validator.herokuapp.com/
